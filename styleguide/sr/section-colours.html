3 Colours Colours

Toggle full screen Open in new window Toggle example guides Toggle HTML markup

These are the default colours

For font class, use .type--{{modifier_class}}

For background class, use .bg--{{modifier_class}}

Example

$black: #1e1e1e;

$dark-grey: #282828;

$jasper-grey: #666;

$greyboy-grey: #999;

$gainsboro-grey: #ccc;

$item-grey: #ddd;

$light-grey: #eee;

$smoke-grey: #fafafa;

$cw-grey: #f0f0f0;

$white: #fff;

$red: #f0544a;

$berry: #e63f52;

$coral: #f9a567;

$purple: #A25EB5;

$blue:#00C2DF

$green: #A3D55F

$aqua: #75E1C2

$teal: #00A9B0;
